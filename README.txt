

This directory contains the codes for implementing RORL-DDPG for the 
last example in the Numerical Results section. 

Environment: an uncertain optimization problem with driving factor 
space of dimension 10.

Requirements: 
Note: Versions stated are the versions used.

TensorFlow  2.2.0
Numpy       1.18.5
Scipy       1.4.1

Usage: Run the following file

python main.py

Output: A plot of sum of rewards for each episode.
